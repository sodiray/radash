## Basic usage

Given an object returns a new object with the keys and values reversed.

```ts
import { invert } from 'radash'

const powersByGod = {
  ra: 'sun',
  loki: 'tricks',
  zeus: 'lighning'
}

invert(powersByGod) // => { sun: ra, tricks: loki, lightning: zeus }
```
